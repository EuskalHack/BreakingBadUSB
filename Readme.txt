Basado em vusb - https://www.obdev.at/vusb/
Este proyecto busca generar un sniffer usb ls basada en el código de vusb e implementada sobre la placa digispark attiny85
DIRECTORIOS
  POC_attach_interrupt0_Arduino_mode          -> Demo de configuración de interrupción en attiny85
  POC_attach_interrupt_on_USB_pin             -> Demo de configuración de interrupción USB
  POC_connect_disconnect_keyboard             -> Demo de configuración para desconectar dispositivo USB por código
  POC_light_Sensor_on_Attiny85_reset_pin      -> Empleo de sensor de luz para gestión en el pin de reset
  POC_light_Sensor_using_led_in_reverse_mode  -> Empleo de sensor de luz para gestión en el pin de reset bis
